## Prerequisites

These tools are needed to start developing.

```bash
# nvm
brew install nvm

# node (latest lts version)
nvm install --lts

# yarn
brew install yarn --without-node

# install gitbook
yarn add -g gitbook

# install boilerplate creator globally
yarn add -g create-react-app react-scripts-ts
```

## How to Read

```bash
git clone <repo>
cd repo

gitbook serve
```
